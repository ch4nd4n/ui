# Todo App Specification

## Overview

This is a simple todo application. Users can add, edit, complete, and delete tasks.

## Features

1. Add new todo items
2. Edit existing todo items
3. Mark items as completed
4. Delete items
5. Filter todos (all, active, completed)

## UI

- Built with shadcn-ui components
- Styled using tailwind.css
- Responsive design

## Structure

- Components follow atomic design (atoms, molecules, organisms)
- All file names use kebab-case

## Data

- Each todo item has:
  - id
  - text
  - completed (boolean)

## Pages

- Main page: displays todo list and controls

## Notes

- Simple and clean user experience
- Fast and accessible
